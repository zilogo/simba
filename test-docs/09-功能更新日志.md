## 升级说明

### 从2.4.x升级到2.5.0

1. 备份数据
   ```bash
   ./backup.sh
   ```

2. 拉取新版本
   ```bash
   docker-compose pull
   ```

3. 执行迁移
   ```bash
   docker-compose run --rm web python manage.py migrate
   ```

4. 重启服务
   ```bash
   docker-compose up -d
   ```

如有问题，请联系技术支持。
