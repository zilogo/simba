智云科技产品常见问题解答（FAQ）

========================================
一、账号相关问题
========================================

问：如何注册智云科技账号？
答：您可以通过以下方式注册账号：
1. 访问官网 www.zhiyuntech.com，点击"免费试用"
2. 填写公司名称、联系人、手机号、邮箱等信息
3. 阅读并同意服务协议
4. 点击"提交注册"
5. 通过手机验证码完成注册
注册成功后，您将获得15天免费试用期。

问：一个公司可以注册多少个账号？
答：每个公司可以注册一个主账号，主账号可以创建多个子账号。子账号数量根据您购买的产品版本有所不同：
- 基础版：最多5个子账号
- 专业版：最多50个子账号
- 企业版：不限子账号数量

问：忘记密码怎么办？
答：请按以下步骤重置密码：
1. 在登录页面点击"忘记密码"
2. 输入注册时使用的邮箱地址
3. 查收重置密码邮件（注意检查垃圾邮件文件夹）
4. 点击邮件中的链接，设置新密码
如果仍无法登录，请联系客服400-888-9999。

问：如何修改绑定的手机号？
答：修改绑定手机号需要进行身份验证：
1. 登录系统，进入"账号设置"
2. 点击"安全设置"
3. 在"手机绑定"处点击"修改"
4. 通过原手机号验证身份
5. 输入新手机号并验证
6. 完成修改

========================================
二、产品功能问题
========================================

问：智能客服支持哪些渠道接入？
答：我们的智能客服系统支持以下渠道：
- 网页在线客服
- 微信公众号/小程序
- 企业微信
- APP内嵌客服
- 电话语音客服（需额外购买语音模块）
- 邮件客服
不同版本支持的渠道数量不同，详情请咨询销售顾问。

问：知识库支持哪些文件格式？
答：知识库支持以下文件格式：
- 文档类：PDF、Word（.doc/.docx）、TXT、Markdown
- 表格类：Excel（.xls/.xlsx）、CSV
- 演示类：PowerPoint（.ppt/.pptx）
- 图片类：JPG、PNG、GIF（用于图文知识）
单个文件大小限制为50MB。

问：如何提高智能客服的回答准确率？
答：提高准确率的方法包括：
1. 完善知识库内容，确保覆盖常见问题
2. 为知识添加准确的关键词和同义词
3. 定期查看"未识别问题"列表，补充相关知识
4. 使用"问答对"功能，针对特定问题设置标准答案
5. 定期进行知识库维护，删除过时内容
6. 收集客户反馈，持续优化

问：系统支持多语言吗？
答：是的，我们的系统支持多语言：
- 基础版：支持简体中文
- 专业版：支持简体中文、繁体中文、英文
- 企业版：支持中文、英文、日文、韩文等多种语言
如需其他语言支持，可联系我们进行定制。

三、技术相关问题
========================================

问：系统对浏览器有什么要求？
答：推荐使用以下浏览器的最新版本：
- Google Chrome（推荐）
- Mozilla Firefox
- Microsoft Edge
- Safari
不建议使用IE浏览器，可能会出现兼容性问题。

问：API接口有调用限制吗？
答：是的，不同版本有不同的API调用限制：
- 基础版：1000次/天
- 专业版：10000次/天
- 企业版：无限制
超出限制后，API调用将返回429错误，次日自动恢复。

问：数据是否安全？
答：我们非常重视数据安全：
1. 所有数据传输采用HTTPS加密
2. 数据存储采用AES-256加密
3. 系统通过ISO 27001认证
4. 支持私有化部署
5. 定期进行安全审计
6. 提供数据备份和恢复功能

问：系统支持私有化部署吗？
答：企业版支持私有化部署，可以部署在：
- 客户自有服务器
- 私有云环境（阿里云、腾讯云、华为云等）
私有化部署需要满足以下最低配置：
- CPU：8核
- 内存：32GB
- 存储：500GB SSD
- 带宽：100Mbps

四、付款与发票问题
========================================

问：支持哪些付款方式？
答：我们支持多种付款方式：
- 在线支付：支付宝、微信支付
- 银行转账：对公账户转账
- 信用卡：支持Visa、MasterCard
企业客户推荐使用对公转账。

问：如何申请发票？
答：申请发票流程：
1. 登录系统，进入"费用中心"
2. 选择需要开票的订单
3. 点击"申请发票"
4. 填写发票信息（抬头、税号等）
5. 选择发票类型（电子发票/纸质发票）
6. 提交申请
电子发票即时发送，纸质发票7个工作日内寄出。
