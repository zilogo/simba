# 智能客服系统用户指南

## 第一章：系统登录

### 1.1 登录方式

智能客服系统支持多种登录方式：

1. **账号密码登录**
   - 打开系统网址：https://cs.zhiyuntech.com
   - 输入您的用户名和密码
   - 点击"登录"按钮

2. **企业微信登录**
   - 点击登录页面的"企业微信登录"
   - 使用企业微信扫描二维码
   - 确认授权后自动登录

3. **单点登录（SSO）**
   - 企业版用户可配置SSO登录
   - 通过企业统一认证系统登录

### 1.2 忘记密码

如果您忘记了登录密码，请按以下步骤操作：

1. 点击登录页面的"忘记密码"链接
2. 输入您的注册邮箱
3. 系统将发送密码重置链接到您的邮箱
4. 点击链接，设置新密码
5. 使用新密码重新登录

注意：密码重置链接24小时内有效，请及时处理。

## 第二章：工作台介绍

### 2.1 界面布局

登录后，您将看到智能客服工作台，主要分为以下几个区域：

1. **顶部导航栏**
   - 显示当前登录用户信息
   - 提供系统设置、帮助中心等入口
   - 显示未读消息通知

2. **左侧菜单**
   - 会话管理：处理客户对话
   - 知识库：管理知识内容
   - 数据统计：查看业务报表
   - 系统设置：配置系统参数

3. **中央工作区**
   - 显示当前功能的主要内容
   - 支持多标签页操作

4. **右侧信息栏**
   - 显示客户详细信息
   - 显示相关知识推荐

### 2.2 快捷操作

系统支持以下快捷键：

- Ctrl + Enter：发送消息
- Ctrl + K：打开知识库搜索
- Ctrl + T：转接人工客服
- Ctrl + E：结束当前会话
- Ctrl + N：创建新工单

## 第三章：会话管理

### 3.1 接待客户

当有新客户进入时，系统会自动分配或手动选择客服接待：

1. 新会话会出现在"待接待"列表中
2. 点击会话可查看客户问题
3. 点击"接待"按钮开始服务
4. 会话自动移入"进行中"列表

### 3.2 智能辅助

在与客户对话过程中，系统提供智能辅助功能：

1. **智能推荐回复**
   - 系统根据客户问题自动推荐回复
   - 点击推荐内容可一键发送
   - 也可编辑后再发送

2. **知识库搜索**
   - 输入关键词搜索相关知识
   - 可直接引用知识内容回复客户

3. **快捷回复**
   - 使用预设的快捷回复模板
   - 支持自定义快捷回复

### 3.3 会话转接

当遇到复杂问题需要转接时：

1. 点击工具栏的"转接"按钮
2. 选择转接目标（其他客服或部门）
3. 填写转接备注说明情况
4. 确认转接

### 3.4 结束会话

服务完成后，请及时结束会话：

1. 确认客户问题已解决
2. 点击"结束会话"按钮
3. 选择会话分类和标签
4. 填写服务小结（可选）
5. 确认结束

## 第四章：知识库管理

### 4.1 知识分类

知识库采用树形分类结构：

- 一级分类：如"产品知识"、"售后服务"
- 二级分类：如"产品A"、"产品B"
- 三级分类：如"功能说明"、"常见问题"

### 4.2 添加知识

1. 进入知识库管理页面
2. 选择知识分类
3. 点击"新建知识"
4. 填写标题和内容
5. 设置关键词和标签
6. 点击保存

### 4.3 知识审核

新添加的知识需要经过审核：

1. 提交后进入待审核状态
2. 管理员审核通过后生效
3. 审核不通过会退回修改
